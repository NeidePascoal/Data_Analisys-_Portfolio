# 📊 Data Analysis Portfolio – SQL Server, Python & R

## 📌 Overview

This repository contains a collection of practical exercises developed during my Data Analyst training.
The work is organized into three main areas:

* **SQL Server (SSMS)**
* **Python (VS Code)**
* **R Programming**

Each folder demonstrates key data analysis skills including data cleaning, transformation, visualization, and clustering.

---

## 📂 Repository Structure

```
📁 SQL_Server/
📁 Python/
📁 R/
```

---

## 🗄️ SQL Server (SSMS)

### 🔹 Description

This folder includes exercises developed using Microsoft SQL Server Management Studio (SSMS).

### 🔹 Topics Covered

* Database creation and management
* Data querying using SELECT statements
* Joins (INNER, LEFT, RIGHT)
* Aggregations (SUM, AVG, COUNT)
* Views and stored procedures

### ▶️ How to Run

1. Open **SQL Server Management Studio (SSMS)**
2. Connect to your server
3. Open the `.sql` files
4. Execute queries using **"Execute" (F5)**

---

## 🐍 Python (VS Code)

### 🔹 Description

This folder contains data analysis exercises using Python, developed in Visual Studio Code.

### 🔹 Technologies Used

* pandas
* numpy
* matplotlib / seaborn

### 🔹 Topics Covered

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Data visualization
* Basic data manipulation

### ▶️ How to Run

1. Open the project in **VS Code**
2. Install dependencies:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
3. Run Python files:

   ```bash
   python filename.py
   ```

---

## 📈 R Programming

### 🔹 Description

This folder includes statistical analysis and clustering exercises using R.

### 🔹 Topics Covered

* Correlation analysis
* Data normalization (scaling)
* Data visualization (corrplot, corrgram)
* Clustering (K-means, silhouette method)

### ▶️ How to Run

1. Open files in **RStudio** or R environment
2. Install required packages:

   ```r
   install.packages(c("corrplot", "corrgram", "cluster", "factoextra", "ggplot2"))
   ```
3. Run scripts step by step
